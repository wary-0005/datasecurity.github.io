How to proceed

Click on login button, do login and click on download app button. Download the app.

To encrypt the file enter code and click on open button, then choose file you want to encrypt , click on open. Click OK.

Your file is successfully encrypted !! Remember the code you entered to decrypt the file again.

To decrypt the file enter the same code which you've used while encrypt and click on open button, then choose file you want to decrypt , click on open. Click OK.

Your file is successfully decrypted !!
